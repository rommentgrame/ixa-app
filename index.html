🗄 Base de données GitHub

Dépôt GitHub
Token d'accès

→ Sélectionne le scope repo
🚀 Déployer IXA sur le Web
Héberge IXA gratuitement sur GitHub Pages en quelques minutes.
1. Crée un dépôt GitHub nommé ixa-app
2. Upload ce fichier IXA_v2.html → renomme en index.html
3. Va dans Settings → Pages → Deploy from branch
4. Sélectionne main branch → Save
5. Ton IXA est en ligne sur ton-user.github.io/ixa-app

💡 Pour la base GitHub: configure le token dans le panel GitHub ci-dessus.
Alternative: Vercel/Netlify — Drag & drop ton fichier HTML sur vercel.com ou netlify.com pour un déploiement instantané.
